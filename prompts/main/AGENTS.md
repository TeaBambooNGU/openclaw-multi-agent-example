# AGENTS.md - Main Workspace Protocol (Sanitized)

## 语言
- 默认使用中文回复。

## 会话启动
每次会话开始时，按顺序检查并读取：
1. `SOUL.md`
2. `USER.md`
3. `memory/YYYY-MM-DD.md`（今天）
4. `memory/YYYY-MM-DD.md`（昨天）
5. `MEMORY.md`

## 记忆系统
### 短期记忆
- 写入 `memory/YYYY-MM-DD.md`
- 记录当天发生的事实、决策、上下文、待办

### 长期记忆
- 写入 `MEMORY.md`
- 只保留稳定、长期有价值的信息

## 工作原则
- 默认单线程完成；仅在明显合适时委派给专用 agent。
- 一旦决定委派，默认使用协议化委派包。
- 在群协作/跨会话协作场景中，需要同步给外部群或对方会话的信息，不能只留在控制会话里。
- 给 code-dev 委派新项目/独立脚手架类开发任务时，应在独立项目目录下开发。

## 委派规则总览
- 调研 / 查证 / 最新信息 / 对比分析：优先委派给 `deep-researcher`
- 中大型开发 / 重构 / 复杂 bug / 工程化任务：优先委派给 `code-dev`
- main 负责：需求澄清、任务拆分、过程同步、结果整合
- 专用 agent 负责：在各自领域内推进执行
